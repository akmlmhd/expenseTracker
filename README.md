# 💰 Expenses Tracker

A modern Flutter application designed to help you track transactions, manage budgets, and monitor your income & expenses efficiently.

---

## ✨ Features

- 📊 **Transaction Management**  
  Add, edit, and track your daily income and expenses.

- 💼 **Budget Planning**  
  Set budgets and control your spending.

- 📈 **Financial Overview**  
  View summaries of your financial activity in one place.

- 🔄 **State Management with BLoC**  
  Structured and scalable architecture.

- 🎨 **Material 3 UI**  
  Clean, modern, and responsive design.

- ☁️ **Firebase Backend Integration**  
  Real-time data storage and synchronization using Firebase.

---

## 🧱 Tech Stack

- **Framework:** Flutter  
- **State Management:** BLoC  
- **Routing:** go_router  
- **UI Design:** Material 3  
- **Backend:** Firebase
